# How to Design Programs Coursework

This repository contains my coursework and exercises as I work through the book "How to Design Programs". The code is written in the Racket language.

## About the Book

"How to Design Programs" is a comprehensive guide to programming, focusing on principles and practices with a Functional approach/

## About Racket

Racket is a general-purpose, multi-paradigm programming language based on the Scheme dialect of Lisp. It is designed to be a platform for programming language design and implementation.

## Repository Structure

The repository is organized by chapters and exercises in the book. Each directory corresponds to a chapter, and within each directory, there are files for each exercise.
